InitToolsContextProvider

Context provider that initializes MarvinJS and RDKit.
Until tools are in loading state, shows spinner that prevents any user interaction.
If tools failed to load for some reason, shows an error toast message by calling enqueueSnackbar from ToastProvider.

Name Type Description
children ReactNode Components that needs an access to the context

// app should be wrapped once at its highest level
