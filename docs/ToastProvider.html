Namespace: ToastProvider

ToastProvider

Context provider that gives an ability to call toast messages to all the child components.
To show toast message, call enqueueSnackbar function.
Properties:
Name Type Description
children ReactNode Components that needs an access to the context

Examples

// app should be wrapped once at its highest level
// to make sure every component gains access to toast messages
<ToastProvider>
  <YourComponent/>
</ToastProvider>
// rendering the toast
// possible variants: error, info, success, warning
enqueueSnackbar({
  message: "Your message",
  variant: "info",
});
